# Five-session usability plan

**Duration:** ~35–40 minutes each (including setup). **Order:** Counterbalance VR-first vs desktop-first between participants.

## Session outline

1. **Intro (5 min)** — Purpose, consent, comfort in headset.
2. **Warm-up (5 min)** — Create two nodes and one link in **both** environments (order counterbalanced).
3. **Main task (15 min)** — Same structured prompt in each environment (e.g. plan a small project with 8–12 nodes, explicit links).
4. **Recall probe (5 min)** — Distraction task, then sketch or list the map from memory.
5. **Debrief (8 min)** — Semi-structured interview using [templates/observation-sheet.md](templates/observation-sheet.md).

## Environment pairing

- **A:** Spatial Mind Canvas (WebXR on Quest Browser) vs **B:** A desktop mind-mapping tool the participant already knows, or a simple 2D canvas with boxes and lines.
- Swap order across participants.

## Facilitator checklist

- [ ] HTTPS URL ready; device charged.
- [ ] Onboarding path available; allow skip if needed.
- [ ] Screen recording optional; never required for participation.

## Stop conditions

Stop the XR portion if the participant reports notable discomfort — continue debrief on desktop only.
