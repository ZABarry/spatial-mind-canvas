# Task scripts — spatial vs desktop comparison

Use the **same** creative prompt in both conditions. Adjust wording for your audience; keep complexity comparable.

## Prompt A — project plan

“Outline a small software feature: stakeholders, risks, milestones, and open questions. Use **8–12 nodes** and show **relationships** with links (not only a list). Spend up to **12 minutes**.”

## Prompt B — research synthesis

“You’re preparing a short brief from three sources (invent names). Capture themes, disagreements, and follow-ups. **8–12 nodes**, linked.”

## Stop rules

- Facilitator calls “time” at 12 minutes; participant may finish the current action.
- If stuck >2 minutes on one interaction, offer **one** neutral hint (e.g. “try Recenter if you’ve lost the selection”).

## Post-task (before recall)

“Point to where you put the single most important idea.” (Note gesture vs panel path.)
