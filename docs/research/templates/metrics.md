# Lightweight metrics

Rate where indicated on **1–5** unless noted.

| Metric | How to capture | Notes |
|--------|----------------|-------|
| Time to first meaningful map | Stopwatch from “start task” to ≥3 nodes + 2 links | Define “meaningful” in the task script. |
| Task completion | Boolean + partial credit | e.g. required nodes/edges present. |
| Ease (SUS-like) | 1–5 Likert after each condition | Keep the same questions across VR/desktop. |
| Confidence in final map | 1–5 | “I trust this captures my thinking.” |
| Structure recall | Count correct edges/nodes from memory | After distraction step. |
| Flow | 1–5 | “I stayed focused without fighting the tool.” |
| Comfort / friction | 1–5 + free text | Motion, clarity, errors. |
| Repeat-use intent | 1–5 | “I would use this again for a similar task.” |
